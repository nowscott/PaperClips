## 🎨 UI 与架构优化细节 (UI/UX & Architecture Details)
在核心系统还原之外，我们还针对现代 React 架构和更好的玩家体验进行了如下优化与限制：

### 状态管理 (Zustand Slices)
- **Store 拆分**：将原本臃肿的 `gameStore.ts` 拆分为了 8 个独立且职责明确的状态切片（`businessSlice`, `tickSlice`, `manufacturingSlice`, `factorySlice`, `spaceSlice` 等），大幅提升了代码的拓展性和可维护性。
- **循环依赖修复**：在拆分过程中使用 `import type { GameState }` 解决了 Vite 在处理循环依赖时抛出的类型引用错误。

### 界面交互与显示条件
- **指数移动平均 (EMA) 速率统计**：
  - **每秒收益 (Rev/sec)** 与 **每秒销量 (Sales/sec)**：在解锁 **收益追踪器 (RevTracker)** 后显示于商业面板。
  - **每秒产量 (Clips/sec)**：在拥有任何一台**自动制造机 (AutoClippers)** 或**巨型制造机 (MegaClippers)** 时显示于制造面板底部。
- **组件条件渲染**：
  - **自动进货机 (WireBuyer)** 开关：仅在完成 `WireBuyer` 项目后（`hasWireBuyer = true`）才在制造面板出现。
  - **面板解锁**：计算 (Computing) 和 项目 (Projects) 面板在前期被隐藏，只有当回形针总产量（`clips`）达到 2000 时才会出现。项目面板被移动到了制造面板下方，以优化视觉层级。
  - **高级投资风险等级**：投资面板初始只显示 `Low Risk`。只有当投资引擎等级 (Investment Engine Level) 提升至 2 级及以上时，才会依次解锁 `Med Risk` (Lv.2) 和 `High Risk` (Lv.3) 的选项，避免初期信息过载。
- **终端日志 (Console Log) 布局优化**：
  - 默认状态只显示最新一行日志，支持点击展开。
  - 修复了展开后因外层固定高度和 Flexbox 默认 `shrink` 属性导致文本堆叠挤压的问题（使用了 Tailwind 的 `shrink-0` 和 `break-words`）。

### 数值与成本曲线对齐 (Cost Curves Alignment)
为了保证 100% 还原原版《Universal Paperclips》的游戏体验，我们对所有核心实体的价格和成本增长曲线进行了严格的数学公式对齐：
1. **自动制造机 (AutoClippers)**：修复为原版指数加法公式 `Math.pow(1.1, level) + 5`。
2. **巨型制造机 (MegaClippers)**：调整为温和的 `1.07` 指数增长 `Math.pow(1.07, level) * 1000`。
3. **投资引擎 (Investment Engine)**：修复关键机制，升级不消耗算力 (Ops)，而是消耗 **Yomi**，成本曲线为 `Math.floor(Math.pow(level + 1, Math.E) * 100)`。
4. **策略锦标赛 (Tournaments)**：移除成本递增逻辑，恢复为每次固定消耗 **1000 Ops**。
5. **工厂与无人机 (Factories & Drones)**：
   - **回形针工厂**：复刻了原版的阶梯分段衰减系数 `fcmod` (随着等级提升，价格倍率逐渐从 11 倍平滑下降至 1.1 倍)。
   - **采集/加工无人机**：修复为原版的多项式增长公式 `Math.pow(level + 1, 2.25) * 1,000,000`。
6. **太空探测器 (Probes)**：将发射造价彻底锁定为原版终极目标门槛：固定的 **100,000,000,000,000,000 (100 Quadrillion / 10亿亿)** 个回形针。

## 📝 结论与当前任务
目前，**阶段一、阶段二、阶段三（含“创造力”、“巨型制造机”、“策略锦标赛”、“工厂与无人机”）已经 100% 迁移完毕**。

接下来我们的开发核心将集中在游戏最终章：**阶段四 (Late Game - Space & Swarm)** 的剩余收尾工作，具体任务优先级如下：
1. **星际战斗 (Space Combat & Drifters)**：实现叛变探测器（Drifters）的生成逻辑以及基于 Combat 属性的自动战斗推演。
2. **结局与转生系统 (Endgame & Prestige)**：当宇宙探索度达到 100% 后触发剧情对话，允许玩家选择重置游戏并进入“下一个宇宙”或“更高维度的模拟层级”，继承永久加成。
